## Usage:

1. Difference between ***before.json***
```
{
  "host": "hexlet.io",
  "timeout": 50,
  "proxy": "123.234.53.22",
  "follow": false
}
```
   and **after.json**
```
{
  "timeout": 20,
  "verbose": true,
  "host": "hexlet.io"
}
```
